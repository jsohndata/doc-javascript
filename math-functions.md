# Math Functions
```
const numbers = [3, -5, 1.5, 10, 4];
```

```
const ceiling = Math.ceil(3.2); //4
const floor = Math.floor(3.2); //3
const rounded = Math.round(3.2); // 4
const absoluteValue = Math.abs(-10); //10
```

```
const maxNumber = Math.max(...numbers) // 10
const minNumber = Math.min(...numbers) // -5
```

```
const power = Math.pow(2,3) // 8
const squareRoot = Math.sqrt(25); // 5
```

```
const random = Math.random(); // between 0 and 1
```
